# 🎯 H5 应用使用说明

## 访问地址
http://165.154.240.3

## 钱包连接说明

### 前提条件
需要安装 MetaMask 钱包插件：
- Chrome: https://metamask.io/download/
- 移动端: 下载 MetaMask App

### 连接步骤
1. 打开 http://165.154.240.3
2. 点击 "Connect Wallet" 按钮
3. MetaMask 会弹出授权窗口
4. 点击"连接"授权
5. 连接成功后会显示钱包地址

### 测试页面
http://165.154.240.3/test.html
- 可以测试钱包是否正常连接

### 常见问题

**Q: 点击按钮没反应？**
A: 请确保已安装 MetaMask 插件

**Q: 移动端如何使用？**
A: 使用 MetaMask App 内置浏览器打开链接

**Q: 提示"未检测到钱包"？**
A: 需要先安装 MetaMask 浏览器插件
