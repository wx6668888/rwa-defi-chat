## ⚠️ 需要配置云服务器安全组

当前服务器端口 3000 已正确监听，但需要在云服务商控制台开放端口：

### 配置步骤：

1. 登录云服务器控制台
2. 找到安全组/防火墙规则
3. 添加入站规则：
   - 协议：TCP
   - 端口：3000
   - 来源：0.0.0.0/0（所有 IP）

### 常见云服务商配置位置：

**AWS EC2:**
- Security Groups > Inbound Rules > Add Rule

**阿里云:**
- 安全组 > 配置规则 > 添加安全组规则

**腾讯云:**
- 安全组 > 入站规则 > 添加规则

**Azure:**
- Network Security Group > Inbound security rules
